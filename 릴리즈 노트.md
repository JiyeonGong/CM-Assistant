# 릴리즈 노트

## 미배포

- 붙여넣기 출석부에서 외출 상태는 감지되지만 외출 시작/복귀 시간이 보고서에 표시되지 않던 문제를 수정했습니다.
  - `시작 시간`, `복귀 시간` 단독 헤더도 외출 시간 열로 인식합니다.
  - 보고서에는 `김코드(외출 13:00~14:00)`처럼 외출 시간까지 표시됩니다.
- 붙여넣기 표가 헤더 1줄과 데이터 1줄만 있는 짧은 표일 때도 분석할 수 있도록 최소 행 제한을 완화했습니다.
- 내 루틴 관리에서 `수정`이 동작하지 않고 `삭제`할 수 없던 문제를 수정했습니다.
  - 루틴 제목을 인라인으로 수정하고 저장/취소할 수 있습니다.
  - 루틴 삭제 버튼을 추가했습니다.
- 처리할 업무와 완료한 업무가 많을 때 페이지 전체가 길어지지 않도록 각 영역 내부에서 스크롤되도록 개선했습니다.
- 업무 탭의 카테고리 이동 메뉴를 Step 1 `업무 선택` 패널 안으로 옮겨 시선 이동을 줄이고, 업무 목록이 패널 내부에서 스크롤되도록 개선했습니다.
- 업무 탭 카테고리 탭을 스크롤 이동 방식에서 필터 방식으로 변경해 화면 움직임 없이 선택한 카테고리의 업무만 볼 수 있도록 개선했습니다.
- 내 루틴 관리의 추가/수정/사용 여부 변경/삭제 안내를 하단 문구 대신 자동으로 사라지는 toast 알림으로 변경했습니다.
- 캘린더 선택 날짜 상세에서 개인/휴강/공휴일/연차 항목의 컬러 박스 배경을 제거하고, 일반 업무처럼 표시하되 메타 텍스트 색상만 카테고리별로 다르게 보이도록 개선했습니다.
