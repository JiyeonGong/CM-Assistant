# CM Assistant

CM Assistant는 클래스 매니저의 출결 보고, 반복 업무, 운영 멘트 복사를 돕는 로컬 데스크톱 앱입니다.

개인정보가 포함될 수 있는 출석부 파일은 서버로 전송하지 않고, 사용자의 PC 안에서만 분석합니다.

## 주요 기능

- 오늘 출석부 `.xlsx` 파일 분석
- 엑셀 표 붙여넣기 방식 분석
- 오전/오후/최종 출결 보고 멘트 생성
- Slack 굵게/글머리 기호 서식을 고려한 클립보드 복사
- 불시점검, 로그 대조 결과 안내, 강사 공유 등 운영 멘트 템플릿
- 불시점검 복귀 확인 타이머와 시스템 알림
- 오늘 업무 대시보드
- 반복 루틴 업무 자동 생성
- 월간 캘린더 기반 업무/일정 확인
- Todo 로컬 저장

## 개인정보

- 출석부 분석은 로컬 앱 내부에서만 실행됩니다.
- 출석부 파일과 Todo 데이터는 외부 서버로 전송하지 않습니다.
- 앱 데이터는 Electron `userData` 경로의 `cm-assistant-data.json`에 저장됩니다.

## 출결 분석 기준

- 집계 대상은 `훈련생 상태`가 `훈련중`인 수강생입니다.
- `중도탈락`, `조기취업` 등 `훈련중`이 아닌 상태는 전체 인원과 출결 집계에서 제외합니다.
- 지각은 입실 시간이 `09:11` 이상인 경우로 분류합니다.
- 조퇴는 퇴실 시간이 있고 `18:50` 이전인 경우로 분류합니다.
- 외출은 출결상태가 `외출`이거나 외출 시작/복귀 시간이 있는 경우로 분류합니다.
- 오전 보고에서는 입실 시간이 없는 사람을 `부재중`으로 표시합니다.
- 오후 보고에서는 입실 시간이 없는 사람을 결석에 포함합니다.
- 최종 보고에서는 입실 시간 또는 퇴실 시간이 없는 사람을 결석에 포함합니다.
- 최종 보고의 `출석입력요청 검토 결과`는 매니저가 직접 확인할 수 있도록 기본값 `0건 / 이상 없음`으로 출력합니다.

## 사용 방법

### 출결 보고

1. `출결` 탭을 엽니다.
2. 오늘 출석부 `.xlsx` 파일을 선택하거나 엑셀 표를 붙여넣습니다.
3. 분석을 실행합니다.
4. 오전/오후/최종 보고 버튼 중 필요한 보고를 선택합니다.
5. `복사하기`를 눌러 Slack에 붙여넣습니다.

### 업무 도구

1. `업무` 탭을 엽니다.
2. 필요한 업무 카테고리와 템플릿을 선택합니다.
3. 내용을 확인하거나 수정합니다.
4. `복사하기`로 Slack, Discord, Zep 등에 붙여넣습니다.

### 오늘 업무 관리

1. `오늘` 탭에서 오늘 할 일을 확인합니다.
2. 루틴 업무는 체크박스로 완료 처리합니다.
3. 추가 업무는 상단 입력창에서 바로 등록합니다.
4. `관리` 탭에서 월간 캘린더와 전체 Todo를 확인합니다.
